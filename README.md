# 깃허브 브라우저 샘플 앱 (작성중)

간단한 기능을 수행할 수 있는 깃허브 브라우저 앱

## 1. 개발환경

* IDE : Android Studio 3.0.1
* Language : Java

### 사용 라이브러리

* Retrofit2
* ButterKnife
* Dagger2
* Realm
* Glide
* Android Architecture Components (Lifecycle Extensions)
* OkHttp
* Gson

## 2. 지원기능

- [ ] 사용자 프로필 조회
- [ ] 사용자 gist 조회
- [ ] 사용자 gist 내용 출력
- [ ] 사용자 gist 댓글 출력
- [ ] 사용자 팔로워, 팔로잉 조회
- [ ] 저장소 조회
- [ ] 저장소 파일 탐색 / 파일 내용 출력
- [ ] Dagger2 적용
